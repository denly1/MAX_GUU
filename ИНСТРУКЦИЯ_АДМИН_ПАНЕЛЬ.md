## Вход в админ-панель

### Как стать администратором

Администратор назначается одним из двух способов:

1. **Через переменную окружения** (bootstrap-админ):
   - Ваш MAX user_id добавляется в файл `.env` в переменную `ADMIN_IDS`
   - Пример: `ADMIN_IDS=119941670,987654321`
   - После перезапуска бота вы автоматически получаете права администратора

2. **Через существующего администратора**:
   - Существующий админ заходит в админ-панель
   - Выбирает **"👤 Управление администраторами"** → **"➕ Добавить администратора"**
   - Вводит ваш MAX user_id
   - Вы получаете уведомление о назначении

### Как узнать свой user_id

Отправьте боту команду `/id` или `/myid` — бот покажет ваш MAX user_id.

### Доступ к админ-панели

После получения прав администратора:
1. Нажмите `/start` в боте
2. В главном меню появится кнопка **"Админ-панель"**
3. Нажмите на неё для входа в админ-панель

## Главное меню админ-панели

После входа в админ-панель вы увидите следующие разделы:

### Управление пользователями
- **✓ Верификация** — подтверждение или отклонение новых пользователей
- **👤 Управление администраторами** — добавление/удаление администраторов
- **👥 Просмотр пользователей** — просмотр всех зарегистрированных пользователей
- **🔄 Переключение роли** — временное переключение на роль студента/преподавателя/партнёра для тестирования интерфейса

### Управление контентом
- **📋 Проекты** — управление проектами (добавление, редактирование, удаление)
- **📝 Заявки** — просмотр заявок от преподавателей и партнёров
- **🎭 Мемы** — управление мемами (кодовые слова для отправки картинок)
- **📞 Управление контактами** — редактирование контактов администраторов

### Коммуникация
- **📢 Рассылка** — массовая рассылка сообщений пользователям
- **📞 Созвон** — организация созвонов с командами

### Отчёты и статистика
- **📊 Статистика** — просмотр статистики по пользователям и командам
- **📥 Экспорт** — выгрузка данных в Excel

## Управление пользователями

### 1. Верификация пользователей

**Назначение**: Подтверждение регистрации новых пользователей.

**Как использовать**:
1. Нажмите **"✓ Верификация"** в главном меню админ-панели
2. Вы увидите список пользователей, ожидающих подтверждения
3. Для каждого пользователя доступны действия:
   - **"✓ Подтвердить"** — пользователь получает доступ к боту
   - **"✗ Отклонить"** — регистрация отклоняется

**Что видит пользователь**:
- При подтверждении: получает доступ к функциям бота
- При отклонении: получает уведомление об отказе

---

### 2. Управление администраторами

**Назначение**: Добавление и удаление администраторов системы.

#### Добавление администратора

1. Нажмите **"👤 Управление администраторами"**
2. Нажмите **"➕ Добавить администратора"**
3. Введите MAX user_id пользователя (пользователь должен быть зарегистрирован в боте)
4. Пользователь получит уведомление о назначении администратором

#### Удаление администратора

1. Нажмите **"👤 Управление администраторами"**
2. Выберите администратора из списка
3. Нажмите **"❌ Удалить [ФИО]"**
4. Администратор будет понижен до роли студента
5. Пользователь получит уведомление об удалении из администраторов

**Важно**: Нельзя удалить себя из администраторов.

---

### 3. Просмотр пользователей

**Назначение**: Просмотр всех зарегистрированных пользователей с группировкой по ролям.

**Как использовать**:
1. Нажмите **"👥 Просмотр пользователей"**
2. Вы увидите статистику:
   - 🎓 Студенты
   - 👨‍🏫 Преподаватели
   - 🤝 Партнеры
   - 👤 Администраторы
   - ❓ Без роли

---

### 4. Переключение роли

**Назначение**: Временное переключение на другую роль для тестирования интерфейса.

**Как использовать**:
1. Нажмите **"🔄 Переключение роли"**
2. Выберите роль:
   - 🎓 Студент
   - 👨‍🏫 Преподаватель
   - 🤝 Партнер
   - 👤 Администратор
3. Вы увидите интерфейс выбранной роли
4. Для возврата в режим администратора отправьте команду `/admin`

**Важно**: Ваша роль в базе данных временно изменится. Используйте `/admin` для возврата.

## Управление контентом

### 1. Управление проектами

**Назначение**: Создание, редактирование и удаление проектов для студентов.

#### Добавление проекта

1. Нажмите **"📋 Проекты"** → **"Добавить задачу"**
2. Введите данные проекта:
   - Название проекта
   - Описание
   - Социальный партнёр
   - Образовательная программа
   - Количество мест в команде
3. Проект появится в списке доступных для студентов

#### Просмотр и редактирование проектов

1. Нажмите **"📋 Проекты"** → **"Список задач"**
2. Выберите проект из списка
3. Доступные действия:
   - **"Активировать/Деактивировать"** — включить/выключить проект
   - **"❌ Удалить"** — удалить проект

#### Экспорт проектов в Excel

1. Нажмите **"📋 Проекты"** → **"Выгрузить задачи в Excel"**
2. Бот отправит файл Excel со списком всех проектов

---

### 2. Управление заявками

**Назначение**: Просмотр заявок на проекты от преподавателей и партнёров.

**Как использовать**:
1. Нажмите **"📝 Заявки"**
2. Вы увидите список всех поданных заявок
3. Для каждой заявки доступна информация:
   - ФИО заявителя
   - Роль (преподаватель/партнёр)
   - Описание проекта
   - Контактные данные

---

### 3. Управление мемами

**Назначение**: Добавление кодовых слов для отправки картинок.

**Как использовать**:
1. Нажмите **"🎭 Мемы"**
2. Для добавления мема:
   - Нажмите **"Добавить/изменить мем"**
   - Введите кодовое слово (например, `привет`)
   - Отправьте картинку
3. Теперь при отправке кодового слова бот будет отвечать этой картинкой

#### Удаление мема

1. Нажмите **"🎭 Мемы"**
2. Выберите **"Удалить [кодовое слово]"**

---

### 4. Управление контактами администраторов

**Назначение**: Редактирование контактной информации администраторов, которая отображается пользователям.

#### Добавление контакта

1. Нажмите **"📞 Управление контактами"** → **"Добавить контакт"**
2. Введите данные:
   - ФИО
   - Должность
   - Телефон (формат: `8 999-999 99 99`)
   - Email
3. Загрузите фотографию администратора

#### Редактирование контакта

1. Нажмите **"📞 Управление контактами"**
2. Выберите **"Редактировать [ФИО]"**
3. Выберите, что хотите изменить:
   - ФИО
   - Должность
   - Телефон
   - Email
   - Фото

#### Удаление контакта

1. Нажмите **"📞 Управление контактами"**
2. Выберите **"Редактировать [ФИО]"**
3. Нажмите **"❌ Удалить контакт"**

## Коммуникация

### 1. Рассылка

**Назначение**: Массовая отправка сообщений пользователям.

**Как использовать**:
1. Нажмите **"📢 Рассылка"**
2. Выберите аудиторию:
   - Все студенты
   - Все преподаватели
   - Все партнёры
   - Все пользователи
   - Конкретная команда
3. Введите текст сообщения
4. (Опционально) Прикрепите изображение
5. Подтвердите отправку

**Важно**: Рассылка отправляется всем пользователям выбранной категории.

---

### 2. Созвон

**Назначение**: Организация созвонов с командами.

**Как использовать**:
1. Нажмите **"📞 Созвон"**
2. Выберите аудиторию:
   - Конкретная команда
   - Все студенты
   - Все пользователи
3. Введите информацию о созвоне:
   - Дата и время
   - Ссылка на созвон
   - Описание
4. Пользователи получат уведомление с кнопками:
   - **"Буду присутствовать"**
   - **"Не буду присутствовать"**

**Отслеживание ответов**: Бот автоматически собирает ответы пользователей.

## Отчёты и статистика

### 1. Статистика

**Назначение**: Просмотр общей статистики по пользователям и командам.

**Как использовать**:
1. Нажмите **"📊 Статистика"**
2. Вы увидите:
   - Общее количество пользователей
   - Количество по ролям
   - Количество команд
   - Количество активных проектов

---

### 2. Экспорт данных

**Назначение**: Выгрузка данных о выборе проектов в Excel.

**Как использовать**:
1. Нажмите **"📥 Экспорт"**
2. Бот отправит файл Excel с данными:
   - Фамилия, Имя, Отчество
   - Институт
   - Курс
   - Образовательная программа
   - Группа
   - Роль в команде
   - Команда
   - Наименование выбранного проекта
   - Наименование социального партнера

## FAQ по админ-панели

### Как вернуться в режим администратора после переключения роли?

Отправьте команду `/admin` в боте.

---

### Можно ли удалить себя из администраторов?

Нет, система не позволяет удалить себя из администраторов. Попросите другого администратора сделать это.

---

### Что делать, если пользователь не получил уведомление о верификации?

Проверьте, что пользователь не заблокировал бота. Если бот заблокирован, уведомление не будет доставлено.

---

### Как изменить данные в уже созданном проекте?

Сейчас редактирование проектов не реализовано. Удалите старый проект и создайте новый с правильными данными.

---

### Можно ли отменить рассылку после отправки?

Нет, рассылка отправляется мгновенно всем пользователям. Будьте внимательны при вводе текста.

---

## Техническая поддержка

При возникновении проблем с админ-панелью:
1. Проверьте, что вы являетесь администратором (есть кнопка "Админ-панель" в главном меню)
2. Попробуйте перезапустить бота командой `/start`
3. Обратитесь к разработчику системы
